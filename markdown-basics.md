### Text Formatting
- **Bold text**: `**bold**`
- *Italic text*: `*italic*`
- ~~Strikethrough~~: `~~strikethrough~~`
- `Inline code`: `` `code` ``

### Headers
```markdown
# H1 Header
## H2 Header
### H3 Header
#### H4 Header
```

### Lists
1. Ordered Lists
    - Use numbers: `1.`, `2.`, `3.`

* Unordered Lists
    - Use asterisks `*` or hyphens `-`
    - Support multiple levels
        - With indentation

### Links and Images
- Links: `[Text](URL)`
- Images: `![Alt text](image_path)`

### Tables
| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |

### Blockquotes
> Use `>` for blockquotes
> > Can be nested

### Mathematical Equations
- Inline math: $E = mc^2$
- Display math:
  $$
  f(x) = \int_{-\infty}^{\infty} e^{-x^2} dx
  $$
